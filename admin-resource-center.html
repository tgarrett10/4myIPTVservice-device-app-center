Order Workflow

Primary Status Flow

  • CONTACTED
  • PAYMENT SENT

Daily Operating Routine
  • Load Admin Orders
  • Work all NEW REQUEST rows first
  • Send payment instructions from the detail panel
  • Mark PAID only after payment is confirmed
  • Mark COMPLETED after setup or delivery is finished

Support and Troubleshooting

Basic Support Checklist
  • Confirm the customer is on the correct setup page
  • Verify username, password, and server URL exactly
  • Restart the device and reopen the app
  • Clear app cache if the app is acting unstable
  • Use the support page before deeper troubleshooting
