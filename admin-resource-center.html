Internal Maintenance Tools

Update Checker Use this page during your monthly maintenance review to test live links, check current app versions, and mark items as verified.
  • Open Update Checker
APK Version Tracker Spreadsheet Keep this spreadsheet as the source of truth for versions, dates verified, device compatibility, and notes before updating any page.

APK Library Review Workflow Follow this order every month:
  • Open Update Checker
  • Test APK links from apk-library.html
  • Mark failures and replacements in the spreadsheet
  • Update apk-library.html first, then dashboard.html if needed
Link Audit Notes When you test links, record:
  • App name
  • Link works or fails
  • Error message
  • Replacement URL if found
