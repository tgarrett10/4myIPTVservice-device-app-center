Downloader Portal

How to Use Downloader

1. Open the Downloader app on your device

2. Enter the code provided to you

3. Click Go and allow the file to download

4. Install the app when prompted

5. Delete the file after install to save space

Important Notes

- Make sure Unknown Sources is enabled for Downloader

- Install updates over existing apps unless it fails

- If install fails, delete the old version and try again

- Restart your device if you experience issues
