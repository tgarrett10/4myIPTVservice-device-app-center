Quick Device Setup Checklist

Use this exact order on every device.
  1. Connect device to TV and Wi-Fi
  2. Disable Play Protect scanning
  3. Enable Developer Mode
  4. Install Play Store apps
  5. Install Core APK apps
  6. Install Optional Movie/TV APK apps if needed
  7. Install Optional Sports APK apps if needed
  8. Arrange home screen order
  9. Test app launches and playback
  10. Load IPTV credentials only if service is purchased
