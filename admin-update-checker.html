Best Practice

Update Rules

  • Review every direct APK app monthly.
  • Update the spreadsheet first.
  • Only update the site if the version or link actually changed.
  • Test the APK link from the Admin APK Library page after every change.
  • Then test the matching dashboard button and Downloader Portal if applicable.
  • Exclude Google Play Store apps from this audit unless you decide to replace them with direct APK installs.
Best Practice

Control Notes

  • Update the spreadsheet first.
  • Only update the site if the version or link actually changed.
  • Test the APK link from the Admin APK Library page after every change.
  • Then test the matching dashboard button.
  • If a link breaks, replace it in the Admin APK Library before touching customer pages.
Reminder: A Downloader code is only a shortcut that redirects to a URL. If you want direct access to a page like the Resource Center in Downloader, give that page its own direct URL or its own short code.
