Build Profiles

Build Profiles

Choose the profile that best matches the device build you want to create.
Standard Build: Fully loaded build with IPTV, Play Store apps, and side-loaded apps.
Light Build: IPTV plus Play Store apps only. No side-loaded apps.
Sports Build: Focused on IPTV and sports-related apps.
Play Store Bundle: Play Store apps only. No IPTV and no side-loaded apps.
Recommended build sequence: Play Store apps first, Direct shortcut apps next, Remaining direct APK apps, Open your internal APK Library for any controlled backup installs, arrange the home screen, load IPTV credentials, test playback and deliver.

Device Setup Prior to Install

Device Setup Prior to Install

This section is intentionally placed near the bottom. Use the top button if you need the setup steps before installing.
D1
Turn Off Play Protect Scanning
Start on the ONN device home screen. Press the gear button, select All Settings, go to Privacy, then open Security. Turn off Scan apps with Play Protect and Improve harmful app detection.
D2
Allow Unknown Apps for Downloader
Go to Settings → Apps → Special app access → Install unknown apps. Open Downloader and turn permission On. On some devices this may appear under Privacy or Security.
D3
Enable Developer Options
Go to Settings → System or Device Preferences → About. Scroll to Android TV OS Build or Build. Press OK 7 times until Developer Options are enabled.
D4
Recommended Developer Options
Set Window animation scale, Transition animation scale, and Animator duration scale to 0.5x or Off. Do not change advanced items unless you intentionally need to test them.
D5
Best Install Flow After Setup
After settings are complete, install Downloader first, open the dashboard in Downloader, choose your apps, then use the Selected Workflow section to finish the build in order.
