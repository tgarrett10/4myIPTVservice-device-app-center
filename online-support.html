Required App

You should already have IPTV Smarters Pro installed on your Android-based device before moving to the login steps below.

Step 1

Install IPTV Smarters Pro

Install IPTV Smarters Pro using the appropriate app source available on your Android-based device.
If your device does not support IPTV Smarters Pro directly, contact support before trying other setup methods.
Step 2

Log In with Xtream Codes

Open IPTV Smarters Pro and choose Login with Xtream Codes API.

Enter These Exactly

  • Username
  • Password
  • Server URL

Important

Enter your username, password, and server URL exactly as provided. Even a small typo or extra space can prevent login.

Use copy and paste when possible to avoid mistakes.

Common Problems

Basic Troubleshooting

Wrong Login Type

Make sure you selected Xtream Codes API and not another login method.

Incorrect Details

Double-check the username, password, and server URL for typos, missing characters, or extra spaces.

Restart and Retry

Restart the app, restart your device, and confirm your internet connection is stable before trying again.

Important Service Note

Your login is intended for the service and device setup provided to you. Using the same login on multiple devices at the same time can reduce quality and create playback issues.
