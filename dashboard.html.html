Setup

Device Setup Prior to Install

This section is intentionally placed near the bottom. Use the top button if you need the setup steps before installing.
D1
Turn Off Play Protect Scanning
Start on the ONN device home screen. Press the gear button, select All Settings, then go down to Privacy. Scroll all the way to the bottom and open the last option, Security. Turn off Scan apps with Play Protect, and make sure Improve harmful app detection is turned off as well.
D2
Allow Unknown Apps for Downloader
Go to Settings → Apps → Special app access → Install unknown apps. Open Downloader and turn permission On. If your device menu is slightly different, look for Security or Privacy, then find the unknown apps setting there.
D3
Enable Developer Options
Go to Settings → System or Device Preferences → About. Scroll to Android TV OS Build or Build. Press the remote OK button 7 times until the device says Developer Options are enabled.
D4
Recommended Developer Options
Open Developer Options and look for the three animation settings: Window animation scale, Transition animation scale, and Animator duration scale. Set each one to 0.5x for a cleaner feel, or Off if you want the fastest response. Do not change advanced items like background process limits unless you are testing and know exactly why you need it.
D5
Best Install Flow After Setup
After these settings are complete, install Downloader first, open the dashboard inside Downloader, choose your apps, then use the Selected Workflow section to finish the build in order.
