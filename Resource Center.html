Suggested Resource Center Workflow

Simple process to keep this page useful and organized.
  1. Upload a document to GitHub or another stable location.
  2. Copy the document link.
  3. Edit this Resource Center page and replace an Add Link button with the real URL.
  4. Keep the Installer Dashboard focused on app installs and the Resource Center focused on documents.
  5. Only share this page with trusted helpers or installers.
